
# City_pollution
external API, app that allows you to check the most polluted cities

## Getting Started

To run the project download the respository and lunch index.html file.
You can also check live demo at http://www.igor-olszewski.pl/test/pollution.html

### Prerequisites

Google Chrome or any other browser.

### Installing

* Download respository.
* Launch index.html file.

## Built With

* HTML 5
* CSS3
* vanilla JS

API that I have used:
*  https://docs.openaq.org/.
*  https://www.mediawiki.org/wiki/API:Query.
